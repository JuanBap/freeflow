👋 Welcome to FreeFlow

This app solves the FreeFlow puzzle: connect matching numbers with non‑crossing paths until the board is fully covered.

🧠 How the algorithm works

  • Exploration: extend paths step‑by‑step from each number.
  • Validation: prune moves that block cells, go out of bounds, or cross existing paths.
  • Backtracking: when a branch fails, undo steps and try alternatives (depth‑first search).
  • Success: all pairs are connected and the board has no empty cells.

🕹️ How to use

  • Upload a board file (.txt) or use the examples.
  • Draw by dragging between equal numbers.
  • Use Auto Solve to let the algorithm run.
  • After it finishes, open View Statistics to inspect the search.

Tip: press Esc to close.
